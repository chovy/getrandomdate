# getrandomdate

Get a random date between two dates in JSON format.

# /days-ago/:days

	GET /days-ago/7

Gets a random date from the last 7 days.

# /hours-ago/:hours

	GET /hours-ago/4

Gets a random date from the last 4 hours.

# /seconds-ago/:90

	GET /seconds-ago/90

Gets a random date from the last 90 seconds.
